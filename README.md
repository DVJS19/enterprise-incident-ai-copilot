# Enterprise Incident Response Copilot

An enterprise-grade GenAI platform demonstrating:

- Multi-Agent Architecture
- MCP Tool Calling
- Retrieval-Augmented Generation (RAG)
- AWS Bedrock
- Claude Sonnet
- OpenSearch Vector Search
- Terraform Infrastructure as Code
- AWS Serverless Architecture
- Incident Investigation Workflows

## Problem Statement

Enterprise incident investigations require engineers to manually gather:

- Metrics
- Deployment history
- Previous incidents
- Service ownership
- Runbooks

This project automates evidence collection and AI-assisted root cause analysis.
